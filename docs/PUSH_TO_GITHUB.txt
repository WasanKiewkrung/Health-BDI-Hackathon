วิธี push code ลง GitHub repo

ผม push ให้ตรง ๆ ไม่ได้ถ้าไม่มีสิทธิ์หรือ GitHub token ของเจ้าของ repo
ให้ทำตามนี้ในเครื่องตัวเอง:

1) แตก zip นี้ แล้วเข้าโฟลเดอร์ usemed_real_line_rich_v8_structured

2) เปิด Terminal / Git Bash:
   git init
   git branch -M main
   git remote add origin https://github.com/WasanKiewkrung/Health-BDI-Hackathon.git
   git add .
   git commit -m "Add USE MED real LINE rich menu PHP MySQL AI system"
   git push -u origin main

ถ้า repo มีไฟล์อยู่แล้วและ push ไม่ได้ ให้ใช้:
   git pull origin main --allow-unrelated-histories
   git add .
   git commit -m "Merge USE MED system"
   git push origin main

ถ้า GitHub ถาม username/password ให้ใช้ Personal Access Token แทน password
